# Raw Reference Data & Old Projects

This directory contains the original backup and raw files used to generate the ESTUN digital library. We keep this here so that future engineers can access the raw configurations, study the old project examples, and re-generate the JSON if necessary.

## Contents
- **`configTable.cfg` & `usertable.cfg`**: The raw LUA/CFG files from the ESTUN controller that define the programming IDE's instruction list and syntax.
- **Python Scripts (`*.py`)**: Scripts used to parse the raw files, translate descriptions (EN to RU/TR), and automatically generate the Excel and JSON files.
- **Excel Files (`*.xlsx`)**: Intermediate categorized manuals generated during the parsing process.
- **`Examples_Old_Projects/`**: Complete backups of old ESTUN programs and files. Very useful for referencing how certain commands are actually used in production code.
